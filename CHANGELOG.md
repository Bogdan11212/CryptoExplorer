
# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Multi-chain blockchain explorer supporting 6 networks (BTC, ETH, BNB, TRON, TON, LTC)
- Real-time market data integration with Coinlore API
- Block explorer with detailed block information
- Transaction tracking and search functionality
- Wallet address lookup and balance tracking
- Universal search with auto-detection
- Responsive design with mobile-first approach
- Dark/Light theme support
- Apple-inspired glassmorphism UI design
- Network switching capabilities
- Copy-to-clipboard functionality for addresses and hashes
- Top wallets ranking for each network
- Network statistics dashboard

### Technical
- React 18 + TypeScript frontend
- Express + TypeScript backend
- shadcn/ui component library
- TanStack Query for state management
- Tailwind CSS for styling
- Wouter for routing
- PostgreSQL support via Drizzle ORM
